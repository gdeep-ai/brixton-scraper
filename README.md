Goal: Help a client test job-site scraping to extract all role data into an Excel file. 
A recruiter will then run a candidate profile through my custom GPT using that spreadsheet to identify additional roles where the candidate could realistically fit. 
The model will focus on the core requirements needed to succeed in each role and filter out secondary skill noise. 
This gives staffing agencies more opportunities to cross-submit candidates, increase placement volume, and improve speed to placement for contract roles.
